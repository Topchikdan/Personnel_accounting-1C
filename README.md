# Кадровый учёт 1С (HR-1C)

Конфигурация 1С (учёт кадров организации), выгруженная в исходные файлы и версионируемая через git.

## О проекте

Конфигурация автоматизирует базовый кадровый учёт: ведение организационной структуры (отделы, должности), карточек сотрудников и истории их кадровых перемещений — от приёма на работу до увольнения.

Исходники выгружены штатным механизмом платформы (**Конфигурация → Выгрузить конфигурацию в файлы...**) в текстовом/XML-виде, что позволяет вести историю изменений в git без хранилища конфигурации (актуально для базовых версий 1С, где хранилище недоступно).

## Состав конфигурации

### Справочники

| Справочник | Назначение |
|---|---|
| **Сотрудники** | Карточки сотрудников организации |
| **Должности** | Перечень должностей |
| **Отделы** | Структурные подразделения (отделы) |
| **ПолСотрудника** | Справочник пола сотрудника |

### Документы

| Документ | Назначение |
|---|---|
| **ПриемНаРаботу** | Оформление приёма сотрудника на работу |
| **Увольнение** | Оформление увольнения сотрудника |
| **КадровоеПеремещение** | Перевод сотрудника между отделами/должностями |
| **КадровыеПриказы** | Кадровые приказы по сотрудникам |

### Регистры

| Регистр | Назначение |
|---|---|
| **КадроваяИстория** | Регистр сведений, хранящий историю кадровых изменений по сотруднику (приём, перемещения, увольнение) |

### Отчёты и обработки

| Объект | Назначение |
|---|---|
| **СписокРаботающихСотрудников** | Отчёт по текущему списку работающих сотрудников |
| **ОргСтруктура** | Просмотр/обработка организационной структуры |
| **Отчёты** | Общая папка/группа отчётов конфигурации |

## Требования

- Платформа 1С:Предприятие 8.3 (для открытия и редактирования конфигурации в Конфигураторе)
- [OneScript](https://oscript.io) — при использовании пакетного менеджера `opm` и вспомогательных утилит

## Как обновить репозиторий после изменений в конфигурации

1. Откройте базу в режиме **Конфигуратор**.
2. Выполните **Конфигурация → Выгрузить конфигурацию в файлы...**, указав эту же папку.
3. Зафиксируйте изменения:

```bash
git add .
git commit -m "Описание изменений"
git push
```

## Как развернуть конфигурацию из репозитория

1. Создайте новую информационную базу в 1С (или используйте существующую).
2. В Конфигураторе выполните **Конфигурация → Загрузить конфигурацию из файлов...**, указав путь к этой папке.
3. Обновите конфигурацию базы данных (**F7**).
